 ⏰ DeadClock

A modern and responsive frontend task management web application built during the **GDG FrontSprint Hackathon**. DeadClock helps users organize daily tasks and track their progress through a clean, interactive interface.

 🚀 Features

* 📅 Interactive calendar
* ✅ Daily task tracking
* 📊 Progress visualization
* 📱 Responsive design

 🛠️ Tech Stack

* HTML
* CSS
* JavaScript

 🏆 Competition

Developed as part of the **GDG FrontSprint Competition** to showcase frontend development skills, UI/UX design, and responsive web design.
